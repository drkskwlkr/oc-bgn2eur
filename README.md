# OpenCart BGN to EUR Price Converter

Автоматизиран инструмент за преизчисляване на цените на продукти в OpenCart от български лев (BGN) в евро (EUR) във връзка с приемането на еврото в България на 1 януари 2026 г.

## Изисквания

- PHP 7.4 или по-нова версия
- Инсталация на OpenCart
- Достъп до база данни
- Препоръчителен е достъп чрез команден ред (за избягване на проблеми с timeout)
